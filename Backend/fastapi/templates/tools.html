Manual Upload Session

No session

Bulk-add files to a title without filling a form for each one. Pick a title from your library, start a session, then simply forward the files to a channel — each file is attached to that title automatically. Great for adding extra qualities or whole seasons.

How it works
  1. Search by title (with or without year) or an IMDb / TMDB id or link, then select. Titles not yet in your library are added automatically from the forwarded files.
  2. Confirm the options for that title (see the two types below).
  3. Click Start session.
  4. Forward the files to the right channel.
  5. Click End session when you're done.
TMDB / IMDb titles

Nothing to fill in — season, episode and quality are read from each file's name/caption. Forward to your auth or manual channel. If your files use absolute episode numbers with no season (e.g. anime), set the optional Fallback season.

Personal titles (hand-made)

These files have no metadata, so you set it here: season is required for TV (episode & quality optional). Forward to your manual channel only. With an episode set, every file joins that one episode as a new quality; leave episode empty and each file becomes the next episode.

Configure your auth and manual channels in Settings.

Bot Admin Manager

Idle

Add all your bots as admins to the channels they serve — using your session account. It shows which bot is already an admin in each channel, checks whether your session account is allowed to add admins there, then promotes the ones that are missing with a safe permission set (post / edit / delete / invite — never "add admins" or "change info"). Global-search channels are intentionally left out — your session account reads those directly, so the bots don't need admin there.
